OGame, rewritten as a Go server with a terminal client. The galaxy is run from the command line; this web surface exists for sign-up and alliance management.

Get the client

$ curl -fsSL https://raw.githubusercontent.com/cobanov/terminal-army-go/main/install.sh | bash
$ tarmy

Quick start

  1. Run tarmy and sign in from the browser URL it prints.
  2. Return to the terminal when the sign-in page confirms the session.
  3. Build mines, research tech, dispatch fleets with slash commands.
  4. Join or found an alliance with friends.
