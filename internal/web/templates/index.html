OGame, rewritten as a Go server with a terminal client. The galaxy is run from the command line; this web surface exists for sign-up and alliance management.

Get the client

$ go install github.com/cobanov/terminal-army-go/cmd/tarmy@latest
$ tarmy play --server {{.PublicURL}}

Quick start

  1. Sign up for an account.
  2. Pick a universe and a starting planet from the TUI.
  3. Build mines, research tech, dispatch fleets.
  4. Join or found an alliance with friends.
