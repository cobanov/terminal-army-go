What is this tldr

A terminal-native multiplayer space strategy game in the spirit of OGame. You play from your terminal via the tarmy CLI. Resources accrue in real time, multiple players share the same universe, and the command line is the cockpit.

Install one terminal, two commands

1. Install the CLI:

curl -fsSL https://terminal.army/install.sh | sh

2. Start playing:

tarmy

No config needed for the public shard. The CLI defaults to {{.PublicURL}}, opens a browser sign-in URL, then returns control to your terminal once authenticated.
