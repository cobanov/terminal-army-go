TERMINAL.ARMY

command a galactic empire from your terminal

A terminal-native, multiplayer space-strategy game in the spirit of OGame. Mines tick in real time, one shared universe, and the command line is your cockpit.

▸ start playing how it works
tarmy — commander@terminal.army 
$ tarmy
  connected as commander Zorthak · universe s1
tarmy> overview
  Homeworld 3:41:8   M 1.24M  C 812K  D 143K   +2,140 e
tarmy> build metal-mine
  ▸ queued Metal Mine L12 — done 15:06
tarmy> galaxy 2:88
   4  Zorthak   [VOID]   ● online
   7  Kessler   — raided 2h ago
tarmy> fleet attack 2:88:7 cruiser=40
  ▸ fleet #12 en route — ETA 01:44
tarmy> █

1 · install the CLI

curl -fsSL https://terminal.army/install.sh | sh

2 · launch into the public universe

tarmy

No config needed. The CLI defaults to {{.PublicURL}}, opens a browser sign-in, then hands control back to your terminal.
